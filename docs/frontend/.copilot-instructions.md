# GitHub Copilot Instructions for QE SE Catalyst

## Project Context

This is a React + Vite + TypeScript project following an agentic workflow for AI-assisted development.

## Agentic Workflow Phases

1. **Planning**: Architecture and component design (`agentic-workflow/instructions/frontend/planning-prompts.md`)
2. **Implementation**: Code generation (`agentic-workflow/instructions/frontend/implementation-prompts.md`)
3. **Refinement**: Optimization and quality (`agentic-workflow/instructions/frontend/refinement-prompts.md`)
4. **Testing**: Test generation (`agentic-workflow/instructions/frontend/testing-prompts.md`)

## Code Generation Principles

- **TypeScript First**: Always use strict TypeScript typing
- **Single Responsibility**: Each component has one clear purpose
- **Unidirectional Data Flow**: Props down, callbacks up
- **Vite Optimization**: ESM modules, tree shaking, HMR-friendly code
- **Testing**: Generate tests alongside implementation

## Key Patterns

- Functional components with hooks
- Custom hooks for reusable state logic
- Service layers for API interactions
- Comprehensive error handling
- Accessibility-first components

## When Asked to Generate Code

- Reference the appropriate prompt template from `agentic-workflow/instructions/frontend/`
- Follow the established architecture patterns
- Generate TypeScript interfaces before implementation
- Include error handling and loading states
- Create corresponding tests

## File Organization

- Components: Single responsibility, typed props interfaces
- Hooks: Reusable state logic with cleanup
- Services: API interactions with error handling
- Types: Comprehensive interfaces and type definitions
- Tests: Component, hook, and service test coverage
